# Data Structures & Algorithms — Python Solutions

*A personal collection of ~44 Python solutions to data‑structures and algorithms exercises, from a university DSA course.*

![Python](https://img.shields.io/badge/Python-3-blue)
![Topic](https://img.shields.io/badge/topic-Algorithms-red)

## Overview

A study collection of clean Python implementations solving classic algorithm and data‑structure problems across several homework sets. Some solutions include experimental timing analysis and matplotlib plots to verify algorithmic complexity empirically.

## Topics Covered

- **Sorting:** merge sort, quicksort
- **Recursion & number systems:** Fibonacci, base‑K conversion, decimal/binary
- **Strings:** rotation, pattern matching
- **Data structures:** stacks, lists, arrays
- **Combinatorics:** Pascal's triangle, Fibonacci sequences
- **Search & games:** binary search, number‑guessing
- **Parsing:** HTML tag matching with a stack
- **Complexity analysis:** empirical timing of list indexing, dict and set operations
- **Date logic:** leap years, calendar generation

## Project Structure

```
作业1/          # Homework set 1  — basics: string rotation, Fibonacci, Pascal's triangle, calendars
作业2/          # Homework set 2  — part_a/part_b, includes performance benchmarking
作业3/          # Homework set 3  — e.g. stack-based HTML tag validation
数算小组/       # Study-group exercises (voting logic, game strategies, midterm)
1/             # Misc: merge sort, number guessing, date utilities
2400094622-努尔斯/   # Additional individual solutions
count.py       # Standalone exercise
```

*(Folder names are in Chinese: 作业 = homework, 数算小组 = algorithm study group.)*

## How to Run

Each file is a self‑contained script:

```bash
python 作业1/1.py
```

## Notes

- All solutions in Python 3; comments are a mix of Chinese, Russian and English.
- Pure coursework — no external dependencies beyond `matplotlib` for the timing‑analysis plots.
